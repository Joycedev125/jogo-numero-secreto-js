# Jogo do Número Secreto

Aplicação web interativa desenvolvida com JavaScript onde o jogador deve adivinhar um número secreto gerado aleatoriamente pelo sistema. O jogo fornece dicas informando se o número digitado é maior ou menor que o número correto, até que o usuário acerte.

Este projeto foi criado com o objetivo de praticar conceitos fundamentais de lógica de programação e desenvolvimento web.

---

## Funcionalidades

* Geração de número aleatório
* Validação do palpite do usuário
* Dicas "maior" ou "menor"
* Contagem de tentativas
* Mensagens dinâmicas na interface

---

## Conceitos praticados

Durante o desenvolvimento foram aplicados os seguintes conceitos:

* Variáveis e constantes
* Estruturas condicionais (if / else)
* Laços de repetição
* Funções
* Operador ternário
* Concatenação de strings
* Manipulação do DOM

Funções nativas utilizadas:

* Math.random()
* parseInt()
